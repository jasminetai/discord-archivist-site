FAQ

I can't add my account!

Check:

  • Are you using the command correctly? For instance, ,add Archivist would add an account named "Archivist". Capitalization doesn't matter, but double-check your spelling.
  • Are you joining the verification lobby while logged in with the specified account? Note that clicking the join link isn't enough; you need to be in the game lobby for Archivist to verify you.
  • Are you joining the verification lobby quickly enough? The provided link expires 60 seconds after the command is issued.

I can't add my clan!

Check:

  • Are you using the command correctly? For instance, ,addclan CLAN would add the clan "CLAN". Note that addclan is a single word.
  • Are you the leader of the clan? If not, you won't be able to add the clan. Only the clan leader can add a clan to Archivist.
  • Have you added your account using the add command first?

How do I invite Archivist to my server?

Archivist is kept as a private bot, meaning that there is no public invite link for the bot. Scaling the bot up and obtaining official bot verification from Discord isn't currently planned due to the extent of the changes that doing so would require, which would include migrating the infrastructure to support slash commands and securing more resources for the bot. Unfortunately, this limits the number of servers Archivist can join. Consequently, we tend to deny requests for adding Archivist to inactive servers or servers unrelated to Super Bandit RIP.

If you are still interested in adding Archivist to your server, reach out to @Dreamery in the official Bandit RIP Discord.

Why is Archivist insulting me?

You've probably upset Archivist in some way. We can always reset the bot, but it's a pretty stubborn bot.

Your best bet is probably to apologize.
